## ğŸ”§ Process Followed
1. **Data Cleaning**
   - Removed duplicates, corrected inconsistencies, formatted dates and amounts.
2. **KPI Calculation using Pivot Tables**
   - Total sales, avg rating, avg order value, rating count, total orders.
3. **Feature Creation**
   - Added `Quarter`, `Day`, `Week Number`, and calculated averages to meet requirements.
4. **Visualization**
   - Built pivot charts for monthly, daily, weekly trends, top cities, food type breakdown, and state-wise performance.
5. **Dashboard Designing**
   - Aligned charts, KPI cards, formatted visuals.
6. **Added Slicers**
   - Month, Category, Restaurant, State for interactivity.
7. **Insight Extraction**
   - Interpreted trends and patterns from dashboard outputs.
